API Documentation

Welcome to the StockClip AI API. This service uses an asynchronous polling architecture to prevent timeouts.

Authentication

All requests to the frontend proxy must include the secret API key in the headers.

Headers: { "Content-Type": "application/json", "X-API-Key": "your-super-secret-key" }

1. Search API (Clips)

POST /api/search

Starts a vision-based search for individual stock clips.

Request Body

{ "query": "Man meditating seashore", "orientation": "landscape", // or "portrait" "quality": "1080" // or "4k", "any" }

Response

{ "success": true, "task_id": "a1b2c3d4-..." }

2. Auto-Pilot (Timeline) API

POST /api/timeline

Analyzes a full script and generates a sequence of video clips.

Request Body

{ "script": "The universe is vast. Black holes are mysterious.", "orientation": "landscape", "quality": "1080" }

3. Task Status Polling

GET /api/status/{task_id}

Check the status of your queued task. Poll this endpoint every 3 seconds until status is "completed".

Response (Processing)

{ "success": true, "task": { "status": "processing" } }

Response (Completed)

{ "success": true, "task": { "status": "completed", "data": [ ... array of video objects ... ] } }
