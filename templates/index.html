StockClip AI क्या है?

StockClip AI एक intelligent stock footage finder है जो AI का उपयोग करके आपके script या query के लिए perfect HD/4K video clips ढूंढता है।

Groq LLM FAISS Search CLIP Vision

कैसे शुरू करें?

1

API Key सेट करें

Menu → API Settings में जाकर अपनी Frontend Secret Key डालें और Save करें। यह key locally आपके browser में safe रहती है।

2

Clips Mode — Single Query

"Clips" tab में जाएं, अपना scene describe करें जैसे "Man hiking in mountains 4K" और RUN INTELLIGENCE दबाएं।

3

Script Mode — Full Timeline

"Script" tab में पूरा script paste करें। AI हर sentence के लिए automatically best matching clip ढूंढेगा।

4

Download करें

Results में आए clips को preview करें और Download button से HD clip save करें।
