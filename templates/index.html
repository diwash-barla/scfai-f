API Specification

StockClip AI Engine

Welcome to the StockClip AI Video Retrieval Platform. This API leverages an Asynchronous Pipeline integrating Groq LLM expansion, FAISS semantic ranking, and CLIP Vision verification.

PRODUCTION BASE URL
https://api.stockclip.ai

Security & Rate Limits

Auth Scope
Frontend/Client Only
Timeout / Expiry
Task TTL: 180s
Max Concurrency
5 Requests / sec
Polling Limit
Min Interval: 3s

Header Requirement: Pass your per-user key via X-API-Key. Exceeding limits will return a 429 Too Many Requests.

Task Lifecycle Pipeline

AI processing follows a strict state machine. Monitor the status field during polling.

[1] queued → Task accepted, awaiting worker.
[2] processing → Groq generating search queries.
[3] fetching → Gathering stock clips from providers.
[4] ranking → FAISS semantic matching running.
[5] vision → CLIP model verifying frame accuracy.
[6] completed → Task done. Data payload attached.

Endpoints & Integration

1. Init Job

POST /api/search 
// Request Payload  
{  
  "query": "Cyberpunk city glowing neon rain",  
  "orientation": "landscape", // "landscape" | "portrait"  
  "quality": "4k" // "1080" | "4k" | "any"  
}  
  
// Response (202 Accepted)  
{ "success": true, "task_id": "550e8400..." }

2. Poll Results & Data Contract

GET /api/status/{task_id}

Note on Partial Results: Currently, the API does not stream partial data. Wait for the completed status.

// Response Schema (200 OK)  
{  
  "success": true,  
  "task": {  
    "status": "completed",  
    "count": 5, // Max 5 clips returned per search  
    "data": [  
      {  
        "clip_id": "px-123456",  
        "download_url": "https://...",  
        "quality_label": "4K",  
        "aspect_ratio": "16:9",  
        "visual_score": 92.5, // Range: 0.0 to 100.0 (CLIP Confidence)  
        "query_used": "neon city rain",  
        "description": "Aerial view of a futuristic city.",  
        "tags": ["cyberpunk", "neon", "cityscape"]  
      }  
    ]  
  }  
}

Error & Retry Policy

  • 401 Unauthorized: Token missing/invalid. Do not retry.
  • 404 Not Found: task_id expired (exceeded 180s TTL) or invalid.
  • 504 Gateway Timeout: AI Backend cold boot. Retry automatically 1 time after a 5-second delay.
